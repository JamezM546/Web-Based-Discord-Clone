# Web Based Discord Clone

A modern web-based Discord clone built with React, Vite, and Tailwind CSS. This project replicates Discord's core functionality including real-time messaging, server management, and user interactions.

## 🚀 Features

- **Real-time Messaging** - Send and receive messages instantly
- **Server Management** - Create and manage servers and channels
- **User Authentication** - Login and registration system
- **Search Functionality** - Search messages and content
- **Summary Features** - Automatic and manual message summaries
- **Responsive Design** - Works on desktop and mobile devices

## 🛠️ Tech Stack

- **Frontend**: React 18, Vite
- **Styling**: Tailwind CSS, Material-UI
- **State Management**: React Context
- **Build Tool**: Vite
- **Package Manager**: npm

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/JamezM546/Web-Based-Discord-Clone.git
   cd Web-Based-Discord-Clone
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
```
